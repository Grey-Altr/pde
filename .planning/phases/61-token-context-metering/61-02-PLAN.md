**Argument handling with backward-compatible fallback:**
```bash
NDJSON="${1:-}"
if [ -z "$NDJSON" ]; then
  echo "[ context window ]"
  echo ""
  echo "  Orchestrator context (~estimated)"
  echo ""
  echo "  No NDJSON path provided."
  echo "  Scope: orchestrator only (not subagents)"

```

**Context window size resolution (runs ONCE at startup via inline Node.js):**
- Resolve `PLUGIN_ROOT` using `CLAUDE_PLUGIN_ROOT` env var or `$(cd "$(dirname "$0")/.." && pwd)`
- Single `node -e` invocation that:
  1. Requires `model-profiles.cjs` from `${PLUGIN_ROOT}/bin/lib/model-profiles.cjs`
  2. Reads `.planning/config.json` to get `model_profile` (default: `balanced`)
  3. Looks up `MODEL_PROFILES['pde-executor'][profile]` to get model tier (default: `sonnet`)
  4. Maps tier to context window: `{ opus: 1000000, sonnet: 1000000, haiku: 200000 }`
  5. Outputs the context window size as a plain integer string
- Fallback: `CONTEXT_WINDOW="${CONTEXT_WINDOW:-1000000}"`

**Header display:**
```bash
echo "[ context window ]"
echo ""
echo "  Orchestrator context (~estimated)"
echo ""
```

**Accumulator loop with visual bar:**
- `TOTAL_TOKENS=0`, `EVENT_COUNT=0`, `BAR_WIDTH=32`
- `tail -F "${NDJSON}" 2>/dev/null | while IFS= read -r line; do`
- Per event: `line_len=${#line}`, `est_tokens=$(( line_len / 4 ))`, accumulate
- Refresh display every 5 events OR if EVENT_COUNT <= 5
- Percentage: `PCT=$(( TOTAL_TOKENS * 100 / CONTEXT_WINDOW ))`, capped at 100
- Visual bar: `FILLED=$(( PCT * BAR_WIDTH / 100 ))`, fill with `#`, pad with spaces
- Display using ANSI cursor repositioning (`\033[4;1H\033[J`):
  ```
    Usage:  {N}%  (~est.)

    [####                            ]

    {TOTAL_TOKENS} / {CONTEXT_WINDOW} tokens (~est.)
    Scope: orchestrator only (not subagents)
  ```
- The "not subagents" scope line MUST be present — TOKN-03 mandates orchestrator-only scope
- All percentage and token count displays MUST include `(~est.)` — TOKN-01/03 mandate approximation labels
- Bar construction uses pure bash string concatenation: while loops building `#` and space characters (no external tools)
