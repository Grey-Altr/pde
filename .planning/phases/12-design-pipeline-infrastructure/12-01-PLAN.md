**Functions to implement (all take `cwd` as first arg):**

1. `stripCommentKeys(obj)` — Recursive. For each key in obj: if key === '_comment', delete it; else if value is object, recurse. Return nothing (mutates in place).

2. `ensureDesignDirs(cwd)` —
   - `designRoot = path.join(cwd, '.planning', 'design')`
   - `fs.mkdirSync(designRoot, { recursive: true })`
   - For each domain in DOMAIN_DIRS: `fs.mkdirSync(path.join(designRoot, domain), { recursive: true })`
   - If `DESIGN-STATE.md` does not exist at designRoot: read `templates/design-state-root.md` from cwd, replace `{date}` with `new Date().toISOString().split('T')[0]`, write to `path.join(designRoot, 'DESIGN-STATE.md')`
   - If `design-manifest.json` does not exist at designRoot: read `templates/design-manifest.json` from cwd, JSON.parse it, call `stripCommentKeys(parsed)`, set `parsed.artifacts = {}`, set `parsed.tokenDependencyMap = {}`, set `parsed.generatedAt = new Date().toISOString()`, set `parsed.updatedAt = parsed.generatedAt`, write as `JSON.stringify(parsed, null, 2)`

3. `acquireWriteLock(cwd, owner)` —
   - Read `DESIGN-STATE.md` from `.planning/design/DESIGN-STATE.md`
   - Find the `### Write Lock` table section
   - Parse existing lock row (non-header, non-comment row in the table)
   - If a lock row exists: parse Expires column as Date; if `new Date(expires) < new Date()`, the lock is stale — remove the row and proceed to acquire; otherwise return `false`
   - Insert a new row: `| {owner} | {ISO timestamp} | {ISO timestamp + WRITE_LOCK_TTL_MS} |`
   - Write the modified content back to DESIGN-STATE.md
   - Return `true`

4. `releaseWriteLock(cwd)` —
   - Read `DESIGN-STATE.md`
   - Remove any data row in the Write Lock table (keep header row and comment row)
   - Write back
   - Return (void, always succeeds)

5. `dtcgToCssLines(tokens, prefix)` —
   - `prefix = prefix || ''`
   - `const lines = []`
   - For each key in Object.keys(tokens): if key starts with '$', skip; if node has '$value' property, push `'  --' + prefix + key + ': ' + node.$value + ';'`; else if node is a non-null object, recurse with prefix `prefix + key + '-'` and concat results
   - Return lines array

6. `generateCssVars(tokens)` — Return `':root {\n' + dtcgToCssLines(tokens).join('\n') + '\n}\n'`

7. `readManifest(cwd)` — Read `.planning/design/design-manifest.json`, return parsed JSON or null if file doesn't exist.

8. `writeManifest(cwd, manifest)` — Set `manifest.updatedAt = new Date().toISOString()`, write to `.planning/design/design-manifest.json` as `JSON.stringify(manifest, null, 2)`.

9. `updateManifestArtifact(cwd, code, fields)` — Read manifest (throw if null), set `manifest.artifacts[code] = Object.assign(manifest.artifacts[code] || {}, fields)`, call writeManifest.

**cmd* wrapper functions (for pde-tools.cjs router):**

Each follows the pattern: receive `(cwd, ...args, raw)`, call the internal function, call `output(result, raw)`.

- `cmdEnsureDirs(cwd, raw)` — calls ensureDesignDirs, outputs `{ created: true, root: designRoot }`
- `cmdManifestRead(cwd, raw)` — calls readManifest, outputs result
- `cmdManifestUpdate(cwd, code, field, value, raw)` — calls updateManifestArtifact with `{ [field]: value }`, outputs updated manifest
- `cmdArtifactPath(cwd, code, raw)` — reads manifest, returns `manifest.artifacts[code].path` or error if not found
- `cmdTokensToCss(cwd, tokensFile, raw)` — reads tokensFile as JSON, calls generateCssVars, outputs the CSS string (raw mode: just the CSS text)
- `cmdCoverageCheck(cwd, raw)` — reads manifest, returns `manifest.designCoverage`
- `cmdLockAcquire(cwd, owner, raw)` — calls acquireWriteLock, outputs `{ acquired: boolean }`
- `cmdLockRelease(cwd, raw)` — calls releaseWriteLock, outputs `{ released: true }`
