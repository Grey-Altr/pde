  <action>
Create hooks/archive-session.cjs — the SessionEnd archiver. This is a CJS hook script that:

1. Reads hook JSON payload from stdin (same pattern as emit-event.cjs)
2. Extracts project root from `hookData.cwd` (fallback: `process.cwd()`)
3. Reads `.planning/config.json` to get `monitoring.session_id` and `monitoring.session_start_ts`
4. Reads the session NDJSON file from `path.join(os.tmpdir(), 'pde-session-' + sessionId + '.ndjson')`
5. Aggregates metrics by parsing each line with try/catch per line (skip corrupt lines):
   - `eventCount`: total parsed lines
   - `agentCount`: lines where `event_type === 'subagent_start'`
   - `changedFiles`: Set of unique `file_path` values from `file_changed` events, report `.size`
6. Counts git commits via `spawnSync('git', ['log', '--since=' + sessionStartTs, '--oneline'])` with cwd=projectRoot, timeout=3000ms, wrapped in try/catch defaulting to 0
7. Computes duration: `new Date(sessionEndTs) - new Date(sessionStartTs)` formatted as `Xm Ys` (or just `Ys` if under 1 minute, or `unknown` if either timestamp missing)
8. Generates ISO filename: `new Date().toISOString().replace(/:/g, '-').replace(/\..+$/, '') + '-' + sessionId + '.md'`
9. Creates `.planning/logs/` directory via `fs.mkdirSync(logDir, { recursive: true })`
10. Writes markdown summary to `.planning/logs/{filename}` with this exact template:

```markdown
# PDE Session Summary

**Session ID:** {session_id}
**Date:** {YYYY-MM-DD}
**Duration:** {Xm Ys}

## Metrics

| Metric | Value |
|--------|-------|
| Total events | {N} |
| Agents spawned | {N} |
| Files changed | {N} |
| Commits made | {N} |

## Phase / Plan Progress

No phase/plan events recorded this session.

## Notes

*Raw NDJSON: {ndjson_path}*
*Generated: {ISO_timestamp}*
```

The "Phase / Plan Progress" section is a placeholder — Phase 62 (EVNT-04) will add semantic workflow events. For now, always output "No phase/plan events recorded this session."

**Critical error handling rules:**
- Outer try/catch around everything — `process.exit(0)` in all paths
- If config.json is unreadable, use sessionId='unknown', sessionStartTs=null
- If NDJSON file doesn't exist, write summary with eventCount=0 and note "NDJSON file not found"
- If git fails, commitCount=0
- NEVER throw unhandled — this is a Claude Code hook
- Use `os.tmpdir()` not `/tmp` — macOS returns `/var/folders/...`

Make executable: `chmod +x hooks/archive-session.cjs`
