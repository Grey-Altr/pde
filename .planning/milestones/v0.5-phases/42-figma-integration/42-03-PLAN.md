**workflows/wireframe-figma-context.md:**

Create a sub-workflow that fetches Figma design context and returns it for the wireframe workflow to use as reference.

```xml
<purpose>
Fetch Figma design context (component structure, variable names, layout patterns) from the connected Figma file via MCP and return it as reference context for wireframe generation. Called as a sub-workflow from wireframe.md. Gracefully degrades when Figma is not connected or file URL is not configured. Implements FIG-02.
</purpose>
```

Steps:
1. **Check Figma Connection** — Load connections, check figma.status === 'connected' and fileUrl exists. If not, return empty context with note: "Figma context unavailable — wireframe will use PDE design tokens only."
2. **Look Up Tool Name** — `b.call('figma:get-design-context', {})` to get toolName. Wrap in try/catch.
3. **Fetch Design Context** — Instruct Claude to use `{toolName}` with the Figma file URL: "Use tool {toolName} to get design context from the Figma file at {fileUrl}. Return component structure, variable references, and layout patterns."
4. **Return Context** — Output the Figma design context as structured reference data. If the MCP call fails, return empty context with degraded message.

The output should be captured by wireframe.md and used as additional design reference alongside existing PDE design tokens.

**workflows/handoff-figma-codeConnect.md:**

Create a sub-workflow that fetches Code Connect mappings and formats them as a markdown table for the handoff spec.

```xml
<purpose>
Fetch Figma Code Connect mappings via MCP and format them as a component reference table for handoff spec enrichment. Called as a sub-workflow from handoff.md. Gracefully degrades when Figma is not connected, Code Connect is not set up, or the map is empty. Implements FIG-03.
</purpose>
```

Steps:
1. **Check Figma Connection** — Same pattern as wireframe sub-workflow. If not connected, return empty string with note.
2. **Look Up Tool Name** — `b.call('figma:get-code-connect-map', {})` to get toolName.
3. **Fetch Code Connect Map** — Use `{toolName}` with the Figma file URL. The response is `{nodeId: {codeConnectSrc, codeConnectName}}`.
4. **Handle Empty Map** — If response is empty `{}` or null, output note: "No Code Connect mappings found. To set up Code Connect, install @figma/code-connect CLI and publish component mappings from your codebase."
5. **Format Table** — Convert response to markdown table:
```markdown
## Figma Code Connect Mappings

| Figma Node ID | Component | Source Path |
|---------------|-----------|-------------|
| {nodeId} | {codeConnectName} | {codeConnectSrc} |
```
6. **Return Table** — Output the table for handoff.md to append to the handoff spec.
