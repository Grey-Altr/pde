Create `bin/monitor-dashboard.sh` (chmod +x) with these sections in order:

**1. Shebang and constants:**
```bash
#!/usr/bin/env bash
SESSION="pde-monitor"
MIN_COLS=120
MIN_ROWS=30
PLUGIN_ROOT="${CLAUDE_PLUGIN_ROOT:-$(cd "$(dirname "$0")/.." && pwd)}"
```

**2. DEPS-01 — tmux detection:**
```bash
if ! command -v tmux &>/dev/null; then
```

**3. DEPS-02 — Platform-aware install instructions (inside the tmux-missing block):**
- Detect platform via `uname -s`
- Darwin: show `brew install tmux`, set `INSTALL_CMD="brew install tmux"`, check `command -v brew`
- Linux: check `command -v apt-get`, then `dnf`, then `yum`, then `pacman` (in that priority order)
- Each detected package manager: show its install command, set `INSTALL_CMD` and `INSTALLER_AVAILABLE=yes`
- No recognized manager: show manual install URL `https://github.com/tmux/tmux/wiki/Installing`, set `INSTALLER_AVAILABLE=no`
- Unknown platform: same manual URL

**4. DEPS-03 — Consent-gated auto-install (inside the tmux-missing block):**
```bash
if [ "$INSTALLER_AVAILABLE" = "yes" ]; then
  printf "Run '%s' now? [y/N] " "$INSTALL_CMD"
  read -r CONSENT
  if [ "$CONSENT" = "y" ] || [ "$CONSENT" = "Y" ]; then
    eval "$INSTALL_CMD"
    if ! command -v tmux &>/dev/null; then
      echo "Installation may have failed. Please install tmux manually."
      exit 1
    fi
    echo "tmux installed successfully."
  else
    echo "Skipping installation. Run the command above when ready."
    exit 0
  fi
else
  exit 1
fi
```
Close the tmux-missing `fi`.

**5. jq detection (soft warning):**
```bash
JQ_AVAILABLE=true
if ! command -v jq &>/dev/null; then
  JQ_AVAILABLE=false
  echo "Warning: jq is not installed. Pane filtering will be limited."
  echo "Install: brew install jq (macOS) or apt-get install -y jq (Linux)"
fi
```

**6. NDJSON path resolution (Pattern 5 from research):**
```bash
NDJSON_PATH=$(node -e "
  const fs = require('fs');
  const path = require('path');
  try {
    const cfg = JSON.parse(fs.readFileSync(path.join(process.cwd(), '.planning/config.json'), 'utf-8'));
    const sid = cfg.monitoring && cfg.monitoring.session_id || 'unknown';
    const tmpdir = require('os').tmpdir();
    console.log(path.join(tmpdir, 'pde-session-' + sid + '.ndjson'));
  } catch(e) {
    console.log(path.join(require('os').tmpdir(), 'pde-session-unknown.ndjson'));
  }
" 2>/dev/null)
if [ -z "$NDJSON_PATH" ]; then
  NDJSON_PATH="/tmp/pde-session-unknown.ndjson"
fi
```

**7. Terminal size detection (stty size, outside tmux):**
```bash
if [ -t 0 ]; then
  read -r ROWS COLS < <(stty size 2>/dev/null || echo "24 80")
else
  ROWS=0; COLS=0
fi
[ "$COLS" -eq 0 ] 2>/dev/null && COLS="${COLUMNS:-80}"
[ "$ROWS" -eq 0 ] 2>/dev/null && ROWS="${LINES:-24}"
```

**8. TMUX-10 — Nested tmux detection:**
```bash
NESTED=false
if [ -n "$TMUX" ]; then
  NESTED=true
fi
```

**9. Idempotent session check:**
```bash
if tmux has-session -t "$SESSION" 2>/dev/null; then
  if [ "$NESTED" = "true" ]; then
    tmux switch-client -t "$SESSION"
  else
    tmux attach-session -t "$SESSION"
  fi
  exit 0
fi
```

**10. Session creation (detached):**
```bash
tmux new-session -d -s "$SESSION" -x "$COLS" -y "$ROWS"
```

**11. Session options:**
```bash
tmux set-option -t "$SESSION" pane-border-status top
tmux set-option -t "$SESSION" pane-border-format "[ #{pane_title} ]"
tmux set-window-option -t "$SESSION" remain-on-exit on
```

**12. TMUX-09 — Adaptive layout functions:**

Define `build_full_layout()` function:
- Takes `$SESSION` and `$NDJSON_PATH` arguments
- Captures initial pane: `P0=$(tmux display -t "${SESSION}:0.0" -p '#{pane_id}')`
- Split horizontally for right column: `P1=$(tmux split-window -h -dPF '#{pane_id}' -t "${SESSION}:0.${P0}" -p 50)`
- Split P0 vertically: `P2=$(tmux split-window -v -dPF '#{pane_id}' -t "${SESSION}:0.${P0}" -p 66)`
- Split P2 vertically: `P3=$(tmux split-window -v -dPF '#{pane_id}' -t "${SESSION}:0.${P2}" -p 50)`
- Split P1 vertically: `P4=$(tmux split-window -v -dPF '#{pane_id}' -t "${SESSION}:0.${P1}" -p 66)`
- Split P4 vertically: `P5=$(tmux split-window -v -dPF '#{pane_id}' -t "${SESSION}:0.${P4}" -p 50)`
- Label panes: P0="agent activity", P1="pipeline progress", P2="file changes", P3="context window", P4="log stream", P5="token / cost"
- Start pane scripts via send-keys:
  - P0: `bash '${PLUGIN_ROOT}/bin/pane-agent-activity.sh' '${NDJSON_PATH}'`
  - P1: `bash '${PLUGIN_ROOT}/bin/pane-pipeline-progress.sh' '${NDJSON_PATH}'`
  - P2: `bash '${PLUGIN_ROOT}/bin/pane-file-changes.sh' '${NDJSON_PATH}'`
  - P3: `bash '${PLUGIN_ROOT}/bin/pane-context-window.sh'`
  - P4: `bash '${PLUGIN_ROOT}/bin/pane-log-stream.sh' '${NDJSON_PATH}'`
  - P5: `bash '${PLUGIN_ROOT}/bin/pane-token-meter.sh' '${NDJSON_PATH}'`

Define `build_minimal_layout()` function:

- Vertical split: `P1=$(tmux split-window -v -dPF '#{pane_id}' -t "${SESSION}:0.${P0}" -p 50)`
- Label and start only these 2 pane scripts
