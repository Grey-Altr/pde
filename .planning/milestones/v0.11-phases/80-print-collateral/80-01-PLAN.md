The step generates TWO artifacts:

**FLY artifact (always for experience products):**

Generate a self-contained HTML file following the complete A5 scaffold from 80-RESEARCH.md. The file MUST contain:

1. CSS `@page { size: A5 portrait; margin: 0; }` — named size keyword, NOT pixel dimensions for paper formats
2. Format variant switcher with 4 formats: A5 Flyer (148x210mm), A4 Poster (210x297mm), Instagram Square (1080x1080px), Instagram Story (1080x1920px)
3. `body[data-format]` CSS selectors controlling container dimensions per format
4. Bleed zone: `::before` pseudo-element with `inset: -3mm; border: 1.5px dashed rgba(0,80,200,0.5)` — NOT `@page { bleed: 3mm }` which no browser implements
5. Safe zone: `::after` pseudo-element with `inset: 5mm; border: 1px dashed rgba(200,0,0,0.35)`
6. Zone legend below the page container showing bleed and safe zone swatches
7. CMYK approximation table generated from brand colors using the inline `rgbToCmyk` function (12-line formula from research). Table columns: Swatch, Name, Hex, C, M, Y, K, Total. Flag rows where inkTotal > 300% with warning.
8. CMYK disclaimer: "These values are mathematical sRGB-to-CMYK approximations. No ICC profiles or GCR/UCR applied."
9. Mandatory prepress disclaimer `<aside class="prepress-disclaimer">` with text: "COMPOSITION REFERENCE -- NOT A PRODUCTION PRINT FILE. This artifact is a design layout guide. Resolution, color fidelity, bleed registration, and ICC profile conformance have not been verified. Submit production-ready artwork (PDF/X-1a or PDF/X-4, minimum 300 DPI, CMYK, embedded fonts) to your print supplier."
10. `@media print` block hiding format-selector, zone-legend, cmyk-table, prepress-disclaimer, and pseudo-element guides
11. Instagram formats use FIXED px dimensions on `.page-container` with `transform: scale()` — NO `@page` rule for Instagram (they are screen-only, not paper)
12. Print-ready prohibition: Add explicit instruction text: "IMPORTANT: The phrase 'print-ready' is prohibited in generated flyer content. If describing the artifact, use 'composition reference' instead. If the phrase must appear, it MUST be immediately adjacent to the prepress disclaimer block."

Composition annotation block (satisfies PRNT-04):
```html
<!-- COMPOSITION: event-flyer
  Awwwards-level composition rules:
  - Clear focal hierarchy: L1 event name > L2 headliner > L3 date/venue > L4 details
  - Intentional negative space: minimum 25% of page area empty
  - Maximum 3 typefaces: 1 display, 1 body, 1 accent (optional)
  - Grid: rule-of-thirds or golden-ratio — chosen per event aesthetic
  - Color: derived from SYS-experience-tokens.json brand coherence tokens
  - Legible at intended print size (14pt minimum body text at A5)
-->
```
