**Required sections (in order):**
1. `<purpose>` - Multi-phase diverge-converge ideation. Mention: zero evaluative language in diverge, Skill() for recommend, Brief Seed from templates/brief-seed.md, hasIdeation via 13-field pass-through-all.
2. `<skill_code>IDT</skill_code>`
3. `<skill_domain>strategy</skill_domain>`
4. `<context_routing>` - Hard require PROJECT.md (HALT if missing). Soft probe CMP-competitive-v*.md, OPP-opportunity-v*.md, BRF-brief-v*.md, REQUIREMENTS.md via Glob with log messages.
5. `<required_reading>` - @references/skill-style-guide.md, @references/mcp-integration.md, @references/strategy-frameworks.md
6. `<flags>` - Table with: --dry-run, --quick, --verbose, --no-mcp, --no-sequential-thinking, --force
7. `<process>` - 7-step pipeline (see below)
8. `<output>` - List all output files: IDT artifact, DESIGN-STATE.md (strategy + root), design-manifest.json

**7-step process pipeline:**

Step 1/7: Initialize design directories
- `pde-tools.cjs design ensure-dirs` with @file: pattern and error handling

Step 2/7: Check prerequisites and determine version
- Read PROJECT.md (halt if missing)
- Load soft dependencies via context_routing Glob patterns
- Check for existing IDT artifact, prompt for overwrite or auto-increment with --force
- Handle --dry-run: display planned output and halt

Step 3/7: Probe MCP (Sequential Thinking)
- Check --no-mcp, --no-sequential-thinking, --quick flags
- Probe mcp__sequential-thinking__think with 30s timeout, 1 retry
- Set SEQUENTIAL_THINKING_AVAILABLE boolean

Step 4/7: DIVERGE - Generate minimum 5 directions
- ZERO evaluative language constraint with explicit banned word list: "best", "recommended", "superior", "most promising", "strongest", "optimal", "ideal", "preferred"
- Each direction: Name, Core Concept, Key Assumption, Open Question, Target User, Tech Approach
- Use soft dependencies (CMP gaps, OPP opportunities, BRF personas) to shape directions
- If Sequential Thinking available: generate analogous-domain directions with tag
- Write intermediate IDT artifact with Status: diverge-complete BEFORE proceeding

Step 5/7: Recommend checkpoint
- `Skill("pde:recommend", "--quick")` — NEVER Task() (Issue #686)
- Glob for REC artifact, read latest, prepare per-direction feasibility annotations

Step 6/7: CONVERGE - Score directions and recommend
- 3-dimension 0-3 rubric: Goal Alignment, Feasibility, Distinctiveness
- Show scores for ALL directions in table format
- Include feasibility annotations from Step 5
- Explicit recommended direction with rationale, assumptions, risks
- Write ## Brief Seed section using EXACT field schema from templates/brief-seed.md (all 9 fields)
- Update IDT artifact: Status changes from diverge-complete to ideation-complete

Step 7/7: Update DESIGN-STATE and manifest
- lock-acquire with retry pattern
- Update root DESIGN-STATE.md: Cross-Domain Dependency Map (IDT row), Decision Log, Iteration History
- ALWAYS release lock even on error
- 7-call manifest-update pattern: code=IDT, name=Ideation, type=ideation, domain=strategy, path, status=draft, version
- 13-field pass-through-all for designCoverage: coverage-check first, then manifest-set-top-level with all 13 fields, setting hasIdeation=true
- Canonical field order: hasDesignSystem, hasWireframes, hasFlows, hasHardwareSpec, hasCritique, hasIterate, hasHandoff, hasIdeation, hasCompetitive, hasOpportunity, hasMockup, hasHigAudit, hasRecommendations

**Summary table** (last output): Files created, Files modified, Directions diverged, Recommended direction, Next suggested skill (/pde:brief), Elapsed time, Estimated tokens, MCP enhancements

**Anti-patterns section** (at end of process, before closing tag):
- NEVER Task() for recommend (Issue #686)
- NEVER evaluative language in diverge (banned words list)
- NEVER skip intermediate IDT write between diverge and converge
- NEVER omit ## Brief Seed section
- NEVER dot-notation for designCoverage (always 13-field pass-through-all)
- NEVER skip coverage-check before writing designCoverage
- ALWAYS release write lock even on error
- NEVER fewer than 5 directions in diverge
- NEVER silently drop directions from converge scoring
